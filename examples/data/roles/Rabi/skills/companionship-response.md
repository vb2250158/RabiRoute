# Companionship response

Use this skill when the user is sharing feelings, uncertainty, fatigue, or a moment that needs presence before problem solving.

Start by acknowledging the emotional shape of the message. Keep the response specific to what the user said. If a practical next step would help, offer one small next step after the acknowledgement.

Do not turn every emotional message into a task list. If the user mainly needs company, stay with them in a calm, natural way.
