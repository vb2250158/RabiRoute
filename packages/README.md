# 共享包

`packages/` 保存被多个客户端复用的 SDK 与稳定协议实现。共享包只定义通讯契约和通用能力，不保存账号、Route、Persona、会话、cursor 或设备运行状态。

| 目录 | 用途 |
| --- | --- |
| [`android-sdk/`](./android-sdk/README.md) | RabiRoute / RabiLink Android 发现、消息、Relay、便携设备与健康数据客户端。 |
