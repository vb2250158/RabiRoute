# Shared packages

`packages/` contains reusable SDKs and stable protocol implementations. Packages define transport contracts and common capabilities; they do not store account, Route, persona, conversation, cursor, or device-runtime state.

| Directory | Purpose |
| --- | --- |
| [`android-sdk/`](./android-sdk/README_en.md) | Android client for RabiRoute/RabiLink discovery, messaging, Relay, portable devices, and health data. |
