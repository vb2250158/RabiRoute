# Archived local-context implementation

This directory is a read-only migration reference for the removed 0.1 plugin-side role store, binding store, file scanner, and keyword scorer.

It is not a Codex marketplace source and must not return to the active call path. The active plugin under `plugins/rabi-codex-context/` is a thin Rabi PC Manager client; persona, session binding, plan, memory, skill, recall, `viewedAt`, archival, and consolidation behavior belongs to RabiRoute Manager.
